Как пользоваться
  1. Открой index.html (или выложи папку на GitHub Pages).
  2. Меняй вопросы и паки в app.js.
  3. Картинки профилей положи в assets/profiles, клипы — в assets/clips.
  4. Имена/алиасы можно добавлять в поле aliases.
Сделано специально для участников сервера BADDIE.
